🤖 Discipline AI Coach

An AI-powered habit tracking web app that helps users build discipline through daily tracking, streaks, and smart insights.

⸻

🚀 Features
	•	✅ Add & manage habits
	•	🔥 Track daily streaks
	•	📊 Completion analytics (7-day rate)
	•	🤖 AI-powered insights
	•	🗑️ Delete & manage habits
	•	📈 Clean dashboard UI

⸻

🛠️ Tech Stack
	•	Backend: Flask (Python)
	•	Frontend: HTML, CSS, JS
	•	Database: SQLite
	•	Templating: Jinja2

⸻

	•	🔔 Habit reminders
